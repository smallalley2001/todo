About Todo List App

Version 1.0

The Todo List application helps you manage your tasks efficiently. Tasks that are not completed can be edited directly. Completed tasks are locked to prevent accidental changes.

Features:

  • Add, edit, and delete tasks.
  • Tasks are sorted by due date and time; tasks with the same date/time are sorted alphabetically.
  • Save your current task list to a JSON file for backup or sharing.
  • Load a previously saved task list. Note: Loading a list will replace the current tasks.
  • Clear overdue and completed tasks as needed.

Installation as a PWA:

  1. Open the application in your browser.
  2. Look for the "Install" option in your browser menu (or on Chrome/Edge, the "+" icon in the address bar).
  3. Click "Install" to add the app to your desktop or mobile home screen.

Important Notes:

  • Your tasks are stored in the browser's IndexedDB. If you clear browser data, all tasks will be lost.
  • To preserve your tasks, you can use the "Save" button to export them to a JSON file.
  • When loading a JSON file, ensure it is properly formatted and contains the required fields: task, due_date, time_due, priority, comment, completed.
